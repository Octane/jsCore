Изменения в версии 2.8.1 по сравнению с 2.8.0

Версия 2.8.1 вышла 13 июня 2009 года.

    * В callback-функцию метода forEach в качестве 4-го аргумента передается длина массива.

      $.forEach(array, function(element, index, array, length) {…} [, context]);

    * Метод last переименован в getLast.

      $.tag("div").getLast().remove();

    * Метод item переименован в get и при вызове без параметров передает список узлов items.

      $.tag("span").get(0).hide();

    * Исправлена ошибка, возникающая при использовании методов firstChild и lastChild с «пустым» элементом.
    * Методы firstChild, lastChild и nthChild переименованы в first, last и nth соответственно.

      var span = $.tag("div").nth(5, "span");

    * Метод nth (nthChild) больше не принимает 3-й аргумент, позволяющий указать глубину поиска.
    * Удален конструктор core.event и его прототипы.
    * Добавлены следующие методы для работы с объектом-событие:
          o $.target(eventObject) — получить ссылку на DOM-элемент, вызвавший событие.
          o $.relatedTarget(eventObject) — получить ссылку на DOM-элемент, с которого был перемещен указатель мыши.
          o $.mouseButton(eventObject) — определеить, какая клавиша мыши была нажата (1 — левая, 2 — правая, 3 — средняя).
          o $.preventDefault(eventObject) — предотвратить стандартное действие браузера.
          o $.stopPropagation(eventObject) — остановить всплывание события.
          o $.stop(eventObject) — предотвратить стандартное действие браузера и остановить всплывание события.
          o $.pageX(eventObject) — определить позицию курсора мыши по оси X относительно страницы.
          o $.pageY(eventObject) — определить позицию курсора мыши по оси Y относительно страницы.
    * Переписаны тримеры строк, теперь есть всего 3 метода: trim, trimLeft и trimRight — используются встроенные, если возможно.
    * Метод val теперь может получать nodeValue и вызывать $(…).text(string).
    * Метод load быть применен к элементам ввода.

      $('inputID').load(…);

    * Добавлен метод $.isNumber(…).

      typeof NaN == "number" // → true
      $.isNumber(NaN)        // → false

    * Добавлен метод $.computedStyle(node) для получения объекта, содержащего информацию о стилях элемента.
    * Изменена реализация события DOMContentLoaded, теперь js-core может быть подключен после загрузки документа.
    * Метод $.getJSON(…) вместо eval использует JSON.parse, если возможно.
